# Acesso a Banco de Dados com Java

Este projeto e um exemplo simples de programa Java organizado em 4 camadas para introduzir conceitos de persistencia de dados.

Com objetivos didaticos, foram construidas 3 versoes do programa:

- a primeira usa persistencia apenas em memoria, utilizando um `ArrayList`;
- a segunda usa persistencia em banco de dados com JDBC, escrevendo manualmente os comandos SQL;
- a terceira usa ORM com JPA/Hibernate.

Na pasta `docs` ha um tutorial que apresenta em mais detalhes o projeto e cada versao do programa: [docs/tutorial.md](docs/tutorial.md).
